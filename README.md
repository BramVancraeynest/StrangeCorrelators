# StrangeCorrelators

This repo aims to recreate the results of [Mapping topological to conformal field theories through strange correlators [1801.05959]](https://arxiv.org/abs/1801.05959) and is built upon the implementation of symmetric tensor networks provided by [TensorKit.jl](https://github.com/jutho/TensorKit.jl). The diagonalisation uses the Krylov methods provided by [KrylovKit.jl](https://github.com/Jutho/KrylovKit.jl).

Limited testing, to be used with care.
